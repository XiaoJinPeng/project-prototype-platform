## 1. 页面范围

原型页面：`02_原型和PRD/原型/企业端/official_car_bill.html`

页面用途：查看公务车合约账单状态、付款进度和汇款明细。

## 2. 页面口径

1. 公务车账单来自企业合约和配额，不来自单笔员工公务车订单。
2. 员工公务车订单不发起个人支付、个人预授权、个人退款。
3. 账单记录按合约账期、合约车辆数、续费或新增类型生成。

## 3. 数据范围

1. 企业总管理员可查看企业全部账单。
2. 分公司管理员可查看本分公司相关账单。
3. 历史账单按账单生成时快照展示。

## 4. 列表字段

| 字段 | 说明 |
| --- | --- |
| 账单日期 | 账单生成日期 |
| 合约号 | 合约编号 |
| 类型 | 新增、续费 |
| 合约车数 | 本账单覆盖车辆数 |
| 有效期 | 本账单对应服务期间 |
| 账单合计 NT$ | 应付金额 |
| 状态 | 待确认、待汇款、已缴费、已过期 |
| 缴费车数 | 已缴费覆盖车辆数 |
| 缴费金额 | 已缴金额 |
| 操作 | 确认、详情、汇款、汇款明细 |

## 5. 状态与操作

| 状态 | 可执行操作 | 说明 |
| --- | --- | --- |
| 待确认 | 确认 | 企业确认账单内容 |
| 待汇款 | 详情、汇款 | 企业登记汇款资料 |
| 已缴费 | 详情、汇款明细 | 查看缴费完成资料 |
| 已过期 | 详情 | 查看历史账单 |

## 6. 汇款规则

1. 待确认账单确认后进入待汇款。
2. 登记汇款时记录汇款金额、汇款时间、汇款账号、凭证等资料。
3. 平台确认收款后账单进入已缴费。
4. 缴费完成会影响合约续费、有效期和可用配额。
5. 账单过期不自动删除，历史账单只读展示。

## 7. 验收标准

1. 账单页面不展示员工单笔支付信息。
2. 待确认账单可以执行确认。
3. 待汇款账单可以登记汇款资料。
4. 已缴费账单可以查看汇款明细。
5. 历史账单按快照展示合约和金额。
