## 1. 页面范围

原型页面：`02_原型和PRD/原型/企业端/branch_company_management.html`

页面用途：企业总管理员维护分公司资料、管理员账号和分公司可用据点。

## 2. 权限范围

1. 企业总管理员可新增、编辑、停用分公司。
2. 分公司管理员不可新增、编辑、停用分公司。
3. 分公司管理员可查看本分公司基本资料。

## 3. 列表字段

| 字段 | 说明 |
| --- | --- |
| 分公司名称 | 分公司名称 |
| 管理员 | 分公司管理员姓名 |
| 管理员账号 | 分公司管理员登录账号 |
| 可用据点 | 分公司员工可预约公务车的据点，包含平台授权据点和企业自建据点 |
| 员工数 | 分公司员工数量 |
| 部门数 | 分公司部门数量 |
| 状态 | 启用、停用 |
| 操作 | 编辑、停用、启用 |

## 4. 新建分公司弹窗

| 字段 | 必填 | 说明 |
| --- | --- | --- |
| 分公司名称 | 是 | 输入分公司名称 |
| 管理员姓名 | 是 | 输入分公司管理员姓名 |
| Email | 否 | 管理员通知邮箱 |
| 通知管理员 | 否 | 勾选后发送账号通知 |
| 管理员账号 | 是 | 输入管理员登录账号 |
| 管理员密码 | 是 | 输入初始密码 |
| 用车据点 | 是 | 选择该分公司员工可使用的公务车据点，可选范围为已授权给企业的平台授权据点和本企业企业自建据点 |

## 5. 业务规则

1. 用车据点由企业总管理员配置。
2. 可用据点来源包括平台授权据点和企业自建据点。
3. 平台授权据点由营运端授权给企业后，企业总管理员才可分配给分公司。
4. 企业自建据点由企业端建立后，企业总管理员可分配给一个或多个分公司。
5. 新增企业自建据点保存成功后，系统自动加入该据点管理归属分公司的可用据点。
6. 分公司管理员新增本分公司企业自建据点后，只能自动加入本分公司可用据点，不能分配给其他分公司。
7. 分公司员工只能在本分公司可用据点内预约车辆。
8. 分公司停用后，不允许该分公司员工创建新公务车订单。
9. 分公司停用不自动取消历史订单和未完成订单。
10. 分公司已有员工、订单、车辆、账单时，不允许物理删除。

## 6. 验收标准

1. 新建分公司时必须填写分公司名称、管理员姓名、账号、密码和用车据点。
2. 勾选通知管理员后，系统记录通知动作。
3. 分公司可用据点影响新增公务订单的取车据点范围。
4. 分公司可用据点同时支持平台授权据点和企业自建据点。
5. 停用分公司后，该分公司员工不能新建订单。
