## 1. 页面范围

原型页面：`02_原型和PRD/原型/企业端/contract_quotas.html`

页面用途：查看公务车合约、配额、分公司分配和车辆绑定情况。

## 2. 页面口径

1. 页面名称使用“合约配额管理”。
2. 企业端不创建正式合约，正式合约由营运端或平台侧管理。
3. 企业端用于查看合约、分配配额、绑定车辆和确认额度使用。

## 3. 数据范围

1. 企业总管理员可查看企业全部合约与配额。
2. 企业总管理员可向分公司分配合约配额。
3. 分公司管理员可查看本分公司可用合约、配额和绑定车辆。
4. 分公司管理员不可调整其他分公司配额。

## 4. 顶部卡片

| 指标 | 说明 |
| --- | --- |
| 合约数 | 当前查看范围内有效公务车合约数量 |
| 总配额 | 合约下可绑定车辆总数 |
| 已分配配额 | 已分配给分公司的配额数量 |
| 未分配配额 | 总配额 - 已分配配额 |
| 已绑定车辆 | 已占用配额的车辆数量 |

卡片支持点击筛选，再次点击取消筛选。

## 5. 列表字段

| 字段 | 说明 |
| --- | --- |
| 合约编号 | 合约唯一编号 |
| 合约名称 | 合约名称 |
| 合约类型 | 新增、续费等 |
| 计费周期 | 月、季、年等 |
| 有效期 | 合约起止日期 |
| 配额使用 | 已用 / 总配额 |
| 状态 | 有效、待续费、已到期、已终止 |
| 操作 | 详情、分配配额、绑定车辆、调整配额 |

## 6. 配额分配规则

1. 只有企业总管理员可以分配集团合约配额。
2. 分配数量不得超过未分配配额。
3. 调减配额时，调减后的配额不得小于分公司已绑定车辆数量。
4. 合约已到期、已终止或处于不可用状态时，不允许新增分配。
5. 分公司绑定车辆数量不得超过该分公司可用配额。

## 7. 验收标准

1. 总管理员可以查看全部合约配额并分配给分公司。
2. 分公司管理员只能查看本分公司合约配额。
3. 配额不足时，不允许继续绑定新车辆。
4. 合约到期后不再开放新车辆绑定和新预约。
