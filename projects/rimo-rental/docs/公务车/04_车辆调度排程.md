## 1. 页面范围

原型页面：`02_原型和PRD/原型/企业端/dispatch_schedule.html`

页面用途：按车辆和时间查看公务车预约占用情况，用于企业管理员确认车辆排程。

## 2. 数据范围

1. 企业总管理员可查看企业内全部公务车辆排程。
2. 分公司管理员可查看本分公司资产车辆排程。
3. 其他分公司员工预约本分公司资产车辆时，该预约应显示在车辆排程中。

## 3. 时间轴显示规则

| 订单状态 | 时间轴起点 | 时间轴终点 |
| --- | --- | --- |
| 已预约 / 待开始 | 预约取车时间 | 预约还车时间 |
| 用车中 | 实际取车时间 | 预约还车时间 |
| 已完成 | 实际取车时间 | 实际还车时间 |
| 提前还车 | 实际取车时间 | 实际还车时间 |
| 已取消 / 已驳回 | 不显示占用 | 不显示占用 |

## 4. 排程占用规则

1. 已预约和用车中订单占用车辆排程。
2. 已完成订单按实际取还车时间保留历史排程。
3. 提前还车后，时间轴结束时间使用实际还车时间，避免继续显示车辆被占用。
4. 已取消和已驳回订单不占用车辆。
5. 历史排程展示读取订单快照，不受车辆后续调拨、改名、停用影响。

## 5. 页面筛选

支持按以下条件筛选：

1. 分公司
2. 据点
3. 车辆
4. 车款
5. 日期范围
6. 订单状态

## 6. 验收标准

1. 已预约订单在预约取还车时间内显示占用。
2. 用车中订单从实际取车时间开始显示占用。
3. 提前还车订单在实际还车时间后不再显示占用。
4. 已取消订单不显示排程占用。
5. 分公司管理员不能查看无权限车辆的排程管理数据。
