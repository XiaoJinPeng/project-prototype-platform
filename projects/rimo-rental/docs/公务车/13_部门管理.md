## 1. 页面范围

原型页面：`02_原型和PRD/原型/企业端/department_management.html`

页面用途：维护分公司下的部门资料。

## 2. 数据范围

1. 企业总管理员可管理全部分公司的部门。
2. 分公司管理员可管理本分公司部门。
3. 部门用于员工归属、预约审批、用车范围和统计分析。

## 3. 列表字段

| 字段 | 说明 |
| --- | --- |
| 分公司 | 部门所属分公司 |
| 部门名称 | 部门名称 |
| 员工数 | 部门下员工数量 |
| 状态 | 启用、停用 |
| 操作 | 编辑、停用、启用 |

## 4. 新建部门弹窗

| 字段 | 必填 | 说明 |
| --- | --- | --- |
| 分公司 | 是 | 选择部门所属分公司 |
| 部门名称 | 是 | 输入部门名称 |

## 5. 业务规则

1. 部门名称在同一分公司内不可重复。
2. 部门停用后，不允许新员工归入该部门。
3. 部门已有员工时，不允许直接删除。
4. 停用部门不自动调整历史员工、订单和统计快照。
5. 用车管理按部门配置时，部门下新增员工自动继承该部门车辆使用范围。

## 6. 验收标准

1. 新建部门必须选择分公司并填写部门名称。
2. 分公司管理员新建部门时，分公司默认锁定为本分公司。
3. 同一分公司内重复部门名称不允许保存。
4. 停用部门后，员工选择部门时不再出现该部门。
