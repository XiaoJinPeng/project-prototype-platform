| 版本 | 日期 | 调整概括 |
| --- | --- | --- |
| v1.0 | 2026-06-25 | 补建版本记录，作为当前企业端公务车 PRD 基线版本。 |

## 1. 页面范围

原型页面：`02_原型和PRD/原型/企业端/official_order.html`

页面用途：查看和管理企业员工公务车预约订单。

## 2. 数据范围

1. 企业总管理员可查看企业下全部分公司公务订单。
2. 分公司管理员可查看本分公司员工产生的公务订单。
3. 员工跨据点使用其他分公司资产车辆时，订单仍归属于员工所属分公司。

## 3. 列表字段

| 字段 | 说明 |
| --- | --- |
| 订单编号 | 系统生成的公务订单编号 |
| 用车员工 | 员工姓名、手机号 |
| 所属组织 | 员工所属分公司、部门 |
| 取车据点 | 预约取车据点 |
| 还车据点 | 公务车默认同据点归还 |
| 车辆 | 车牌号码、车款 |
| 预约时间 | 预约取车时间、预约还车时间 |
| 实际时间 | 实际取车时间、实际还车时间 |
| 订单状态 | 已预约、待审批、已驳回、用车中、已完成、已取消 |
| 审批状态 | 无需审批、待审批、已通过、已驳回 |
| 操作 | 详情、取消、审批、查看排程等 |

## 4. 订单状态规则

| 状态 | 进入条件 | 可执行操作 |
| --- | --- | --- |
| 待审批 | 员工或管理员提交订单，且员工所属分公司开启预约审批 | 审批通过、驳回、取消 |
| 已预约 | 无需审批直接创建成功，或审批通过后生效 | 取消、查看详情 |
| 用车中 | 已到取车流程并开始用车 | 查看详情 |
| 已完成 | 已完成还车 | 查看详情 |
| 已取消 | 订单被取消 | 查看详情 |
| 已驳回 | 审批被驳回 | 查看详情 |

## 5. 页面规则

1. 页面不展示个人支付、押金、预授权、退款字段。
2. 页面不展示会员方案。
3. 历史订单中的员工、车辆、据点、审批结果按订单快照展示。
4. 当前车辆被调拨、改名、停用后，不反向改变历史订单展示。
5. 订单取消或驳回后，不占用车辆排程。

## 6. 筛选条件

支持按以下条件筛选：

1. 订单编号
2. 员工姓名 / 手机号
3. 分公司
4. 部门
5. 订单状态
6. 审批状态
7. 预约时间
8. 取车据点
9. 车辆

## 7. 验收标准

1. 总管理员进入页面时，可以查看所有分公司订单。
2. 分公司管理员进入页面时，只能查看本分公司员工订单。
3. 用车员工跨分公司据点取车时，订单仍显示在员工所属分公司范围内。
4. 已完成订单展示实际取还车时间。
5. 已取消和已驳回订单不出现在车辆占用排程中。
